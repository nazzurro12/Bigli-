# Base de datos del juego

Coloca aquí contenido editable. El juego lo carga al iniciar.

- `creatures/`: animales, enemigos y civiles.
- `plants/`, `materials/`, `items/`, `buildings/`: contenido del mundo.
- `factions/`, `civilizations/`, `biomes/`, `technologies/`: datos de simulación global.

Los archivos pueden ser paquetes `.json` con categorías o recursos `.tres` de tipo `DFContentDefinition`. Consulta `df_mode/content_packs/README.md` para el formato de criaturas.
