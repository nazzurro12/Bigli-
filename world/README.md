# Generación de mundo visual

Abre `world_generation_lab.tscn` en Godot. Selecciona el nodo raíz **WorldGenerationLab** y usa el Inspector para modificar semilla, nivel del mar, montañas, lluvia, árboles, minerales y cuevas.

La previsualización se actualiza en el editor. Cada cambio se guarda automáticamente en `world_generation_settings.tres`, y cada mundo nuevo usa siempre la densidad de árboles, minerales, cuevas y la semilla elegidas. No modifica una partida ya creada.

Leyenda: azul = agua, beige = desierto, verde claro = pradera, verde oscuro = bosque, dorado = veta rica, violeta = posible cueva y blanco = montaña nevada.
